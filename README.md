# Customer Analysis Dashboard

## Overview

This Power BI dashboard analyzes customer behavior, sales performance, and revenue trends.

## Objectives

- Understand customer segments
- Analyze purchasing behavior
- Monitor sales performance
- Support business decisions

## Tools Used

- Power BI
- Power Query
- DAX
- Excel

## KPIs

- Total Revenue
- Total Customers
- Total Orders
- Average Order Value
- Profit
- Customer Retention

## Visualizations

- Revenue by Category
- Revenue by Region
- Top Customers
- Sales Trend
- Customer Segmentation
- Product Performance

## Key Insights

- Best performing products
- High-value customers
- Revenue growth trends
- Regional sales performance
